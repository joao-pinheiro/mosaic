Mosaic
======
Mosaic is collection of PHP CLI scripts to generate image mosaics, under BSD License.

Get Started
-----------

#### Requirements
Prerequisite packages are:

* PHP 5.5.x/5.6.x
* PHP-Imagick module
* Composer

### Installation
```bash
git clone git://github.com/joao-pinheiro/mosaic.git
cd mosaic
composer update
```
